┌─────────────────────────────────────────────────────────────────────────┐
│ DESKTOP INTERACTIONS                                                    │
└─────────────────────────────────────────────────────────────────────────┘

1. ROTATION (Drag to Rotate)
   ┌─────────────────────────────┐
   │    ← DRAG LEFT → DRAG RIGHT │
   │       Rotates Back    Forward  │
   │    (Reveals card back) (Shows front) │
   │                             │
   │    Releases = Momentum &    │
   │    Inertia (smooth coasting)│
   │                             │
   │    Unlimited spin in any    │
   │    direction!               │
   └─────────────────────────────┘
   
   ├─ Speed: Adjustable via CONFIG.rotation.sensitivity
   ├─ Physics: Momentum-based inertia damping
   ├─ Indicator: Bar at bottom shows progress (0° to 360°)
   └─ Hint: "drag & release anywhere" (disappears after first interaction)

2. ZOOM (Scroll Mouse Wheel)
   ┌─────────────────────────────┐
   │    SCROLL UP = ZOOM IN       │
   │    SCROLL DOWN = ZOOM OUT    │
   │                             │
   │    Range: 0.5x to 3.0x      │
   │    (50% zoom out to 300%)   │
   │                             │
   │    Has momentum! Coasts     │
   │    after you stop scrolling │
   │                             │
   │    Real-time display:       │
   │    "zoom: 1.00x"            │
   └─────────────────────────────┘
   
   ├─ Min: 0.5x (zoom out to see entire card)
   ├─ Max: 3.0x (zoom in for details)
   ├─ Speed: Adjustable via CONFIG.zoom.speed
   ├─ Damping: Smooth coasting animation
   └─ Display: Bottom-left HUD shows zoom level

3. HOVER LIGHTING (Move Mouse Over Card)
   ┌─────────────────────────────┐
   │    🖱️  MOVE CURSOR OVER CARD  │
   │                             │
   │    Within 70% distance:     │
   │    • Gloss reflects cursor  │
   │    • Card glows slightly    │
   │    • Lighting follows move  │
   │                             │
   │    Outside 70% distance:    │
   │    • Effects fade out       │
   │    • Smooth transition      │
   │                             │
   │    Desktop only feature     │
   └─────────────────────────────┘
   
   ├─ Activation: Mouse within ~70% from center
   ├─ Effect 1: Specular gloss reflection
   ├─ Effect 2: Card glow and shadow
   ├─ Intensity: Adjustable via CONFIG.hover.glossIntensity
   └─ Smooth: Gradual fade in/out

4. KEYBOARD CONTROLS (For Power Users)
   ┌─────────────────────────────┐
   │    ← / → or A / D           │
   │    Rotate LEFT / RIGHT      │
   │    (Adds velocity)          │
   │                             │
   │    SPACE or ENTER           │
   │    Gentle flick (toggles)   │
   │    (Reverses if spinning)   │
   │                             │
   │    Tab to focus card first  │
   │    (if not auto-focused)    │
   └─────────────────────────────┘
   
   ├─ ← Arrow: Velocity -= 3 deg/frame
   ├─ → Arrow: Velocity += 3 deg/frame
   ├─ A Key: Same as ← Arrow
   ├─ D Key: Same as → Arrow
   ├─ Space: Gentle flick (±8 deg/frame)
   └─ Enter: Same as Space

┌─────────────────────────────────────────────────────────────────────────┐
│ MOBILE INTERACTIONS                                                     │
└─────────────────────────────────────────────────────────────────────────┘

1. ROTATION (Swipe Left/Right)
   ┌─────────────────────────────┐
   │    ← SWIPE LEFT   │  SWIPE RIGHT → │
   │       Rotates      Rotates       │
   │       Back        Forward        │
   │       (Back face)  (Front face)  │
   │                                 │
   │    Release = Momentum carry      │
   │    (continues spinning)         │
   │                                 │
   │    Single finger tap & drag      │
   └─────────────────────────────────┘
   
   ├─ Method: Touch drag (one finger)
   ├─ Sensitivity: 0.52 degrees per pixel (adjustable)
   ├─ Physics: Momentum-based inertia
   ├─ Works: On all touch devices
   └─ Indicator: Bottom bar shows rotation progress

2. ZOOM (Pinch with Two Fingers)
   ┌─────────────────────────────┐
   │    🤝 PINCH TO ZOOM OUT      │
   │    🤜🤛 SPREAD TO ZOOM IN     │
   │                             │
   │    Two-finger pinch gesture │
   │    Spread fingers to zoom in│
   │    Pinch fingers to zoom out│
   │                             │
   │    Smooth scaling           │
   │    Range: 0.5x to 3.0x      │
   │                             │
   │    Works alongside rotation!│
   └─────────────────────────────┘
   
   ├─ Gesture: Two-finger pinch/spread
   ├─ Zoom: 0.5x (zoomed out) to 3.0x (zoomed in)
   ├─ Sensitivity: ~0.01 per pixel change
   ├─ Display: Real-time zoom level on HUD
   └─ Independent: Works while rotating!

3. DEVICE TILT (Optional - Must Enable)
   ┌─────────────────────────────┐
   │    📱 TILT DEVICE          │
   │                             │
   │    TILT RIGHT               │
   │    → Card rotates FORWARD   │
   │                             │
   │    TILT LEFT                │
   │    → Card rotates BACKWARD  │
   │                             │
   │    Threshold: ±10° tilt     │
   │    Sensitivity: 0.3x        │
   │                             │
   │    (Disabled by default     │
   │     to save battery)        │
   └─────────────────────────────┘
   
   ├─ Feature: Optional (CONFIG.mobile.tiltEnabled)
   ├─ Permission: iOS 13+ requires user approval
   ├─ Sensor: Device accelerometer/gyroscope
   ├─ Axis: Y-axis rotation (gamma)
   ├─ Threshold: ±10° before activation
   └─ Battery: Can be disabled to save power
