⚙️ CUSTOMIZATION (For Developers)

To change settings, open business-card-viewer-v2.html in a text editor
and find the "CONFIG" object near the top.

Example changes:

• Increase rotation sensitivity:
  CONFIG.rotation.sensitivity = 0.75

• Change zoom limits:
  CONFIG.zoom.minScale = 0.3
  CONFIG.zoom.maxScale = 5.0

• Disable hover effects:
  CONFIG.features.hover = false

• Disable accelerometer:
  CONFIG.mobile.tiltEnabled = false

See FEATURES.md for complete config options.

💡 TIPS & TRICKS

1. Zoom in first, then rotate for dramatic effect
2. Move mouse slowly over card for smooth lighting
3. Double-tap to perform "flick" on mobile
4. On mobile, pinch is more intuitive than two-finger scroll
5. Open DevTools (F12) to see zoom level in console
6. The card's position adapts to your viewport size automatically
7. Try tilting your phone (if tilt feature is enabled)
8. Keyboard controls work great for presentations

🐛 TROUBLESHOOTING

Problem: Images not loading
→ Check internet connection (loading from Google Drive)
→ Google Drive files must be publicly shared

Problem: Zoom not working
→ Make sure cursor is over the card
→ Scroll slowly and clearly
→ Check if CONFIG.features.zoom = true

Problem: Mobile gestures not responding
→ Use two separate fingers for pinch
→ Spread/pinch motion must be clear
→ Try on actual device (not simulator)

Problem: Hover effects missing
→ Only works on desktop with mouse
→ Not available on mobile (no mouse)
→ Check CONFIG.features.hover = true
